---

## 🚀 Project Overview

**KeenKeeper** is a modern React-based web application designed to manage and track user activities (timeline, friends, and interactions) in a clean and user-friendly interface.

It focuses on simplicity, performance, and demonstrates advanced React concepts like Context API and client-side data persistence.

---

## ✨ Features

- 🔐 User-friendly interface with responsive design  
- 📊 Timeline system to track activities  
- 👥 Friends management system  
- 🔍 Filtering functionality for better data control  
- 📈 Timeline analysis with interactive **Pie Chart visualization**  
- 💾 Persistent data storage using **Local Storage API**  
- ⚡ Fast performance with optimized React structure  
- 📱 Fully responsive (mobile, tablet, desktop)  

---

## 🛠️ Technologies Used

- **Frontend:** React.js (Vite)  
- **Routing:** React Router  
- **Styling:** Tailwind CSS, DaisyUI 
- **State Management:** Context API  
- **Icons:** React Icons  
- **Storage:** Local Storage API  
- **Deployment:** Vercel
